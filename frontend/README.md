# Hospital Management Frontend

Aplicación React con PWA para gestión hospitalaria.

## Instalación

```bash
npm install
```

## Desarrollo

```bash
npm run dev
```

## Build

```bash
npm run build
```

## Características

- PWA configurado
- Diseño responsive para móvil
- Autenticación JWT
- Gestión de roles (Admin, Enfermero, Paciente)
- Dashboard con estadísticas
- Gestión de islas, camas, pacientes y enfermeros
- Visualización de códigos QR
